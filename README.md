# Vue 3 + Vite

Although the version of vue3.0. but project is use `Vue2` syntax.  Use `vite` develop this project, To improve efficiency. Welcome to use `pnpm` and `vite`.

# Use

- install 

```bash
cd sunway-gis-demo
pnpm install
```

- 运行

```bash
pnpm dev
```

- 打包

```bash
pnpm build
```
